# VT Cloud Bootcamp — Portfolio

Benvenuto nel repository ufficiale del mio percorso da Cloud Engineer.  
Qui documento in modo operativo e professionale tutto ciò che costruisco durante il bootcamp: Linux, AWS, Terraform, troubleshooting e progetti reali.

## 📁 Struttura del repository

- **01-Linux** — esercizi, comandi, troubleshooting, script
- **02-AWS** — servizi, configurazioni, laboratori, scenari reali
- **03-Terraform** — infrastruttura come codice, moduli, progetti
- **04-Troubleshooting** — scenari reali, analisi log, debugging
- **05-Portfolio** — progetti completi e documentati
- **06-Diagrams** — diagrammi architetturali e flussi
- **07-Notes** — appunti tecnici e schemi
## 🎯 Obiettivo del percorso
Costruire un portfolio tecnico completo e dimostrabile, focalizzato su competenze operative richieste nei ruoli di:

- Cloud Support Engineer  
- Cloud Engineer  
- DevOps Junior
